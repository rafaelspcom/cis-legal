# CIS Legal Pages

Páginas legais públicas da **CIS Tecnologia** (CNPJ 30.536.039/0001-99).

Hospedadas via GitHub Pages para garantir acesso público sem proxy/firewall, especialmente para validação automática do Meta Business Manager (WhatsApp Cloud API).

## URLs

Após habilitar GitHub Pages neste repositório:

- **Exclusão de Dados (LGPD Art. 18):** https://rafaelspcom.github.io/cis-legal/exclusao.html
- **Política de Privacidade:** https://rafaelspcom.github.io/cis-legal/privacidade.html (pendente)
- **Termos de Serviço:** https://rafaelspcom.github.io/cis-legal/termos.html (pendente)

## Conteúdo

- Sem código backend, sem credenciais, sem dados de clientes.
- Apenas HTML semântico estático com CSS inline.
- Texto adequado à LGPD (Lei 13.709/2018) e requisitos Meta Business.

## Como habilitar GitHub Pages

1. Settings → Pages
2. **Source:** Deploy from a branch
3. **Branch:** `main` / folder: `/` (root)
4. Save → aguarda 1-2 min

## Manutenção

Atualizações deste site são feitas direto neste repositório.
Conteúdo canônico replicado no CRM principal (privado) em `frontend/public/`.
